# Emberly

A carefully crafted dark theme combining **Ayu Mirage** syntax colors with **GitHub Dark** UI aesthetics.

![Emberly Theme](https://img.shields.io/badge/theme-dark-1e2228?style=flat-square)
![VS Code](https://img.shields.io/badge/VS%20Code-1.60+-007ACC?style=flat-square&logo=visualstudiocode)
![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)

## Variants

| Variant | Background | Style |
|---------|------------|-------|
| **Carbon** | `#22272e` | Balanced mid-tone, GitHub Dark Dimmed |
| **Noir** | `#1e1e1e` | Medium gray, VS Code native feel |
| **Umbra** | `#0d1117` | Darkest, GitHub Dark Default |

## Features

- Full UI theming (editor, sidebar, terminal, panels, status bar)
- Terminal with complete ANSI color palette
- Git decoration colors
- Bracket matching and indent guides
- Input validation colors (error, warning)
- Semantic token support

## Installation

1. Open **Extensions** (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for `Emberly`
3. Click **Install**
4. Select theme: **Preferences: Color Theme** â†’ Choose variant
